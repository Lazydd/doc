# 内置过渡动画

ddlazy-ui 内应用在部分组件的过渡动画，你也可以直接使用。

## fade 淡入淡出

<!-- <CodeRun auto editable> -->

```vue
<template>
	<div>
		<dd-button @click="show = !show">Click Me</dd-button>
		<div style="display: flex; margin-top: 20px; height: 100px">
			<transition name="dd-fade">
				<div v-show="show" class="transition-box">.dd-fade</div>
			</transition>
		</div>
	</div>
</template>

<script>
export default {
	data() {
		return {
			show: true,
		};
	},
};
</script>

<style lang="less" scoped>
.transition-box {
	margin-bottom: 10px;
	width: 200px;
	height: 100px;
	border-radius: 4px;
	background-color: #409eff;
	text-align: center;
	color: #fff;
	padding: 40px 20px;
	box-sizing: border-box;
	margin-right: 20px;
}
</style>
```

## collapse 展开折叠

```vue
<template>
	<div class="fold">
		<dd-button @click="show = !show">Click Me</dd-button>
		<div style="height: 320px">
			<dd-transition>
				<div v-show="show">
					<div class="transition-box">折叠</div>
					<div class="transition-box">折叠</div>
					<div class="transition-box">折叠</div>
				</div>
			</dd-transition>
		</div>
	</div>
</template>

<script>
export default {
	data() {
		return {
			show: true,
		};
	},
};
</script>

<style lang="less" scoped>
.transition-box {
	margin-bottom: 10px;
	width: 200px;
	height: 100px;
	border-radius: 4px;
	background-color: #409eff;
	text-align: center;
	color: #fff;
	padding: 40px 20px;
	box-sizing: border-box;
	margin-right: 20px;
}
.fold {
	.dd-transition {
		margin-top: 20px;
		height: 320px;
	}
}
</style>
```

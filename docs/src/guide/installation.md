# 安装

## npm、pnpm、yarn

::: code-group

```sh [npm]
> npm i ddlazy-ui -S
```

```sh [pnpm]
> pnpm i ddlazy-ui -S
```

```sh [yarn]
> yarn add ddlazy-ui
```

:::

## CDN

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="https://cdn.ddlazy.cn/ddlazy-ui/lib/theme-chalk/index.css" />
<!-- 引入组件库 -->
<script src="https://cdn.ddlazy.cn/ddlazy-ui/lib/index.js" />
```

## JS 运行可编辑代码

:::demo dd-button
<ddButton>11</ddButton>
<ddButton>22</ddButton>
<ddButton>33</ddButton>
:::
